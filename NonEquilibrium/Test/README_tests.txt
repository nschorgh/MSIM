Examples for validation
=======================

Ice sequestration with synthetic temperture profile:

make exper_fast
./exper_fast

This will read input file 'input_fast.par' and produce output files 'z.dat' and
'porefill.dat'.


Ice retreat at Phoenix Landing site over the last 2000 years:

make mars_fast

./mars_fast ph

This will read input file 'lats.ph' and produce output files 'z.ph', 'depths.ph'
and 'depthF.ph'.
